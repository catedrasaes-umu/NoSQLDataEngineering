Eirene: Entity Inheritance Discoverer in Document Stores


UNDER CONSTRUCTION


Description


Eirene (Entity Inheritance Discoverer in Document Stores) is a tool aimed to discover entity inheritance relationships in which a parent entity type is specialized in one or more child entity types. This tool is based on the U-Schema metamodel and its algorithm identifies, based on structural variations of an entity type, existing subtypes and the discriminator field based on certain feature dependencies: Strong dependency, weak dependency and exclusion dependency.
